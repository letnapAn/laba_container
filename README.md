# Laba Container: Custom HashTable Implementation
Учебный проект для демонстрации работы с Maven и Javadoc.

# Возможности
- Интерфейс контейнера
- Два варианта реализации контейнера
  - с сохранением дубликатов (разрешение коллизий - открытая адресация)
  - с игнорированием дубликатов (set реализован методом цепочек)
- Поддержка итератора для for-each
- Полное покрытие кода комментариями Javadoc
- Автоматическая генерация документации через Maven

# Требования
- JDK 17+ (протестировано на JDK 25).
- Maven (встроенный wrapper `mvnw.cmd`).
